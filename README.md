# Java OOP Basics

A simple Java project demonstrating Object-Oriented Programming fundamentals.

## Structure
src/
  └── main/
    └── java/
      ├── Main.java
      └── Student.java

## How to Run
1. Clone this repository
2. Compile: `javac src/main/java/*.java`
3. Run: `java -cp src/main/java Main`
